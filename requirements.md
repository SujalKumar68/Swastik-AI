## 1. Introduction

### 1.1 Purpose
Swastik AI is a chatbot system that helps rural users get basic health guidance based on symptoms.

### 1.2 Scope
The system allows users to:
- Select symptoms
- Get possible diagnosis suggestions
- Chat with AI assistant
- Access emergency support

---

## 2. Functional Requirements

### FR1: Symptom Selection
The system shall allow users to select symptoms from a list.

### FR2: Custom Symptom Input
The system shall allow users to enter symptoms manually.

### FR3: Diagnosis Output
The system shall display possible conditions with confidence scores.

### FR4: AI Chat Support
The system shall provide chatbot-based health assistance.

### FR5: Emergency Button
The system shall provide an emergency assistance feature.

---

## 3. Non-Functional Requirements

### NFR1: Usability
The interface should be simple for rural users.

### NFR2: Performance
The system should respond within a few seconds.

### NFR3: Security
User health data should not be stored without permission.
